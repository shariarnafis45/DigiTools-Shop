# 🛒 DigiTools Shop

DigiTools Shop is a modern and responsive eCommerce web application where users can explore and purchase digital tools easily. The project focuses on clean UI, smooth user experience, and fast performance.

---

## 🚀 Description

This project is built to simulate a digital product marketplace. Users can browse products, view details, and add items to the cart with an interactive and user-friendly interface.

It is designed with a focus on simplicity, speed, and modern frontend practices.

---

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Vite
- React Icons

---

## ✨ Features

- 🛍️ **Dynamic Product Listing**  
  Display digital products with clean UI and responsive design.

- ⚡ **Add to Cart Functionality**  
  Users can add products to cart with instant UI feedback.

- 🎨 **Modern UI/UX Design**  
  Gradient buttons, smooth interactions, and mobile-friendly layout.
